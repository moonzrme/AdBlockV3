# Custom AdBlock V3 Extension

Це просте, але ефективне розширення для блокування реклами, написане під Manifest V3.

## Як встановити:

1. Завантажте або клонуйте цей репозиторій.
2. Переконайтеся, що у вас є всі файли (`manifest.json`, `rules.json`, `content-script.js`, тощо).
3. Відкрийте Google Chrome і перейдіть за адресою: `chrome://extensions/`
4. У правому верхньому куті увімкніть перемикач **"Режим розробника" (Developer mode)**.
5. Натисніть кнопку **"Завантажити розпаковане розширення" (Load unpacked)**.
6. Виберіть папку `AdBlockV3`.

## Як це працює:

1. **Merezhevyi Riven (Network Layer):** `rules.json` використовує API `declarativeNetRequest` для блокування запитів до відомих рекламних доменів (Google Ads, DoubleClick, Analytics) ще до того, як вони завантажаться.
2. **DOM Riven:** `content-script.js` сканує сторінку на наявність рекламних контейнерів (iframe, div з класами ads) і видаляє їх.
3. **Dynamic Monitoring:** `MutationObserver` слідкує за змінами на сторінці і видаляє рекламу, яка підвантажується динамічно (через JS).

## Тестування:

Зайдіть на будь-який сайт з великою кількістю реклами новин або спеціальні тестові сайти для AdBlock, щоб побачити результат.
